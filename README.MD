# Simple To-Do List Application
**Course:** SEN 201

## Description
A simple web-based application to add, view, and delete tasks.
Perfect for demonstrating SDLC.

## Features
- AddTask()
- ViewTasks()
- DeleteTask()

## How to run
1. Open `index.html` in any browser.
2. Type a task in the input box.
3. Click "Add Task" to add it to the list.
4. Click "Delete" to remove any task.

## SDLC
All function and variable names match SDLC documentation.
